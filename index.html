welcome to entangle.

the goal of the game is to fill in the network.

"Slightly open a container"

every puzzle has a theme.

"Slightly open a container" = AJAR

the name of the puzzle encodes the theme.

words are comprised of nodes.

each node holds one symbol.

Controls:

first, click any node to select it and its parent word.

then, use the arrow keys to navigate within the word.

use the keyboard to insert symbols.

press "enter" while an intersecting node is selected
to toggle between words.

press "tab" to cycle through the word list.

press "hint" to reveal the selected node.

press "check" when you're done.
